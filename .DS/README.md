# 数据结构课程设计辅助实现

本目录用于集中展示本项目涉及或可扩展使用的数据结构实现，方便课程设计报告截图与说明。

说明：

- `dataStructures.js` 中的实现是课程设计辅助代码，不属于当前 React/Express 主运行链路。
- 注释中标明了它们在本项目中的对应场景、可替换接入点或同类逻辑位置。
- 当前项目主链路已经实际使用的结构包括：邻接表、逆邻接表、节点映射、边表、集合去重、数组排序、字符串检索、递归图遍历。

推荐报告对应关系：

| 数据结构 / 算法 | 推荐截图位置 | 项目对应场景 |
| --- | --- | --- |
| 线性表 | `LinearList` | `NODES`、搜索结果、时间轴刻度、分类统计 |
| 栈 | `Stack` | DFS 遍历、路径回溯、界面历史状态 |
| 队列 | `Queue` | BFS 遍历、自动播放步骤缓冲 |
| 串 | `KMPMatcher` | 技艺检索、节点名称/简介匹配 |
| 数组与广义表 | `GeneralizedList` | 节点数组、标签二维数组、嵌套分类结构 |
| 树和二叉树 | `BinaryTree` | 遍历生成树、时间区间层级、DOM 树解析 |
| 图 | `TechGraph` | 科技树节点依赖关系、前驱/后继关系 |
| 查找 | `HashSearchTable` | `NMAP` 节点映射、图片资源查找 |
| 排序 | `Sorters` | 按年份排序、按匹配分数排序、按分类数量排序 |
